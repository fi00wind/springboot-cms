
# 一个用springboot为基础做的后台管理系统
##### 文件结构：
###### root
###### -src（后台代码---springboot）
###### -web(前端代码----Vue)

    后台：
        1.使用springboot作为基础框架
        2.使用mysql数据库以及mybatis作为ORM框架
        3.使用shiro作为权限控制器
    前端：
        1.使用Vue作为前端框架
        2.Ant Design of Vue作为UI框架
        3.使用vue-particles添加粒子特效
        
## 目前实现的功能有：
    1.登录、登出
